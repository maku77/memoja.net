`pandas` パッケージは `pip` コマンドで簡単にインストールすることができます。
`pandas_datareader` パッケージを使うと、ネット上から株価データなどを取得することができるので、これもいっしょにインストールしておきます。

```
$ pip install pandas
$ pip install pandas_datareader
```
